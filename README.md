# Project Info

React project for ERP System frontend in typescript

-   setup with vite
-   tailwind css
-   Redux, RTK Query for state management
-   react-chartjs-2 for chart
-   react-router-dom for routing

## Development

To run this project in development mode

```bash
  npm run dev
```
